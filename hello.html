Mes commandes git

pwd *Affiche le chemin du dossier actuel*

mkdir cours_git *Pour créer un dossier "cours_git" dans le dossier actuel*

ls *Liste tous les dossiers et fichiers dans le dossier actuel*

ls cours_git/ *Liste tous les dossiers et fichiers dans le dossier cours_git*

cd .. *pour revenir en arrière d'un dossier*

cd cours_git/ *pour aller dans le dossier "cours_git" qui ce trouve dans le dossier actuel*

git config --global user.name "Lionel Mohr" *Permet de configurer le nom d'utilisateur git de façon globale, pour l'ensemble des créations*

git config --global user.email lionel.mohr@gmail.com *Permet de configurer l'email d'utilisateur git de façon globale, pour l'ensemble des créations*

git config --global --list *affiche la liste des infos git de façon globale, pour l'ensemble des créations*

Git *affiche toutes les commandes de git*

Git --help *affiche toutes les commandes de git, l'aide*

Git version *permet de connaitre la version actuellement installée de git*

Git init CHEMIN_DU_DEPOT *initialise le depot "repository en anglais"*

Git config *permet de configurer Git en local*

Git status *permet d'afficher l'état du dépot actuel*

Git add nom_du_fichier.ext *prend les modif uniquement du fichier concerné*

Git add . *prend les modifs de l'ensemble du dossier*

Git commit -m"Commentaire" *enregistre une modification de l'état des derniers git add*

Git tag *permet de créer un tag à un commit pour passer plus facilement d'un commit à un autre*

Git log *permet d'afficher l'historique des commits*

Git show +sha-1 *permet d'afficher le contenu d'un commit*

Git clone *permet de cloner un dépot* +à revoir

Git push *pouce les commits vers le serveur*

Git push --tags *pouce les tags vers le serveur*

Git push origin --delete BRANCHE_A_SUPPRIMER *Cela supprime la branche à suprimer sur le serveur distant*

Git pull *permet de récupérer les dernier commits d'un serveur distant*

Git checkout *permet de changer de commit ou de branche*

Git pull --rebase *permet de récupérer les dernier commits d'un serveur distant en rebase*

Git branch *liste les branches en local*

Git branch -a *liste les branches sur le serveur*

Git branch -d BRANCH_A_SUPPRIMER *supprime la branche nommé BRANCH_A_SUPPRIMER*

Git switch *à revoir*

Git merge *Permet de faire fusionner deux modifs ensemble*

Git rebase master *Permet de faire un rebase de la derniére branche sur la branche master, il faut être sur le branche que l'on veux rebase par un **git checkout NOM_DE_BRANCHE***

Git help *permet d'afficher l'aide*

Git gui *Permet de lancer l'interface graphique de git*
